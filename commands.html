Shadebot Command Reference


Command Description
General Commands
#daily Allows a user to claim their free daily 100 Shadebucks
#send (@user) (amount) Allows a user to send Shadebucks between players (a fee of 10% is applied on the Shadebucks sent)
#bal [@user] Allows a user to check their own or another user's Shadebucks balance
#roulette (black, red, green) (amount) Allows you to spend your Shadebucks on a game of Roulette. Betting on Green gives you a 1 in 14 chance of winning a large amount, while Black and Red will gives you a 1 in 1.5 chance of winning a smaller amount.
#luckydip (amount) Allows you to spend your Shadebucks on a game of LuckyDip. Use #lb to receive a list of objects and their multipliers via Private Message.

Moderator-Only Commands
#mute (@user) [time] Allows a moderator to mute a specific user (by preventing the user from using chat or voice), the moderator can unmute the user by running the command again or by setting a timer - 1m = the user is muted for 1 minute, 1h = the user is muted for 1 hour. The maximum mute time is currently up to 1 week (168h or 10080m = 1 week).
#roles (ban/unban) (@user) (role) Allows a moderator to impose a ban in order to block a specific user from obtaining a self-assigned role from Shadebot
#settings view [#channel] Allows a moderator to view the permissions Shadebot has to a specific channel.
#welcome Allows a moderator to toggle welcome messages to a Direct Message instead. This is useful when a server receives a large influx of new users.
#library add (library name) (link/-n) Allows a moderator to save links to a library and then randomly post the contents of the library. Using -n instead of a link will create a new library to store links in.
Owner-Only Commands
#botinfo Shows information about the bot
#welcomenum (number) Allows a owner to change the current number displayed on the welcoming counter - this is generally used if something goes wrong with the counter.
#roles add (trigger) (role id) Allows a owner to add a new self-assigned role.
#settings edit (invite/​commands/​image/​link/​reply) (true/false) Allows a owner to change Shadebot's permissions for a specific room, for instance, Shadebot could be configured to disable posting images to a specific room.
#clear (number) Allows a owner to bulk delete messages in a room. The number can be set anywhere from 2 to 100.
